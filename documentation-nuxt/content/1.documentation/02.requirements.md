The project is built with specific tool versions. Please ensure your development environment matches these requirements for optimal compatibility and performance.

## Required Versions

<!-- prettier-ignore-start -->
::card-group
:::card{icon="i-lucide-hexagon" title="Node.js v24+"}
Version **24 or higher** is required to run the project.

[Node.js Official Site](https://nodejs.org/en/){target="_blank"}
:::

:::card{icon="i-lucide-layout-template" title="Next.js 16"}
The project is built with **Next.js 16**.

[Next.js Official Site](https://nextjs.org/){target="_blank"}
:::

:::card{icon="i-lucide-package" title="npm 11"}
Version **11** was used for dependency management.

[npm Official Site](https://www.npmjs.com/){target="_blank"}
:::
::
<!-- prettier-ignore-end -->

## Verify Your Environment

::callout{icon="i-lucide-terminal" color="info"}
To check your installed versions, run the following commands in your terminal:
::

```bash [Terminal]
node -v
npm -v
npx next -v
```

::callout{icon="i-lucide-info" color="neutral"}
Make sure the versions match or exceed the requirements listed above.
::
