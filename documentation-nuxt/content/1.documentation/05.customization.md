The project supports full customization including styling via CSS variables and asset management.

## Design Tokens

Global styles and CSS variables are located in the `src/css` folder. This is where you control the overall look and feel of the application.

![CSS Folder](/images/03.png)

### Edit CSS Variables

```css [src/css/variables.css]
:root {
  *,
  *::before,
  *::after {
    outline: none;
    box-sizing: border-box;
    user-select: none;
    scrollbar-width: none;
    -webkit-user-select: none;
    -moz-user-select: none;
    -ms-user-select: none;
  }
  /* ... */
}
```

## Local Assets

The project contains only one local asset file located at `src/assets/01.png`. All other images are loaded via external URLs defined within the Next.js `Image` component.

## How to Change Images

::steps

### Locate the Image Component

Find the `Image` component in the source code where the image you want to replace is used.

### Replace the URL

Simply swap the existing URL with your own image link:

```tsx
// Before
<Image src="https://old-image-url.com/photo.jpg" alt="description" />

// After
<Image src="https://your-new-url.com/photo.jpg" alt="description" />
```
