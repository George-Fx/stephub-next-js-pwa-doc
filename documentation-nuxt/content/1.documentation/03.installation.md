Before proceeding, verify that your system is ready for project setup.

<!-- STEPS -->

## Steps

::steps

### Open the Project

Open the project folder in a code editor. We recommend **Visual Studio Code**:

::callout{icon="i-simple-icons-visualstudiocode" color="primary"}
[Download VS Code](https://code.visualstudio.com/){target="\_blank"} — free, lightweight, and feature-rich editor.
::

### Install Dependencies

Open the integrated terminal in VS Code (`Ctrl+`` ` or `Cmd+`` `) and run:

```bash [Terminal]
npm install
```

### Start the Project

After installing dependencies, start the development server:

```bash [Terminal]
npm run dev
```

The project will be available at [http://localhost:3000](http://localhost:3000) by default.

::
