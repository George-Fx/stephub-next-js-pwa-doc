The project follows a clean and organized directory structure. Here's an overview of each folder and its purpose.

![CSS Folder](/images/02.png)

## Directory Overview

:::card{icon="i-lucide-cloud" title="src/api/"}
Contains `.json` files that will be fetched via API requests in the future.
:::

:::card{icon="i-lucide-app-window" title="src/app/"}
Contains the main application files, including their associated styles.
:::

:::card{icon="i-lucide-sparkles" title="src/assets/"}
Contains a single asset file used as a decorative element within the application.
:::

:::card{icon="i-lucide-image" title="public/"}
Contains the PWA manifest file with app configuration settings, as well as icons in various sizes required for app installation on different devices.
:::

:::card{icon="i-lucide-blocks" title="src/components/"}
Contains reusable UI components.
:::

:::card{icon="i-lucide-hash" title="src/constants/"}
Contains reusable constants used throughout the application.
:::

:::card{icon="i-lucide-paintbrush" title="src/css/"}
Contains CSS files for styling the application.
:::

:::card{icon="i-lucide-layout-list" title="src/items/"}
Contains item components for different items.
:::

:::card{icon="i-lucide-database" title="src/store/"}
Contains Redux store files for state management.
:::

:::card{icon="i-lucide-pen-tool" title="src/svg/"}
Contains SVG icons implemented as React components for easy integration and customization.
:::

:::card{icon="i-lucide-file-type" title="src/types/"}
Contains TypeScript types.
:::

:::card{icon="i-lucide-wrench" title="src/utils/"}
Contains utility functions and helpers.
:::
